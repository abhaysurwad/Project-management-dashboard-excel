# 📊 Project Management Dashboard – Excel

## 🚀 Overview
This project is an interactive Project Management Dashboard built using Microsoft Excel.  
It helps track project performance, task progress, and key business metrics.

## 📌 Key Features
- Task Progress Tracking  
- Department-wise Performance Analysis  
- Payment Mode Distribution  
- Total Stipend & Budget Overview  

## 🛠 Tools & Skills Used
- Microsoft Excel  
- Pivot Tables  
- Slicers  
- Data Visualization  
- KPI Tracking  
- Advanced Formulas  

## 📈 Insights
This dashboard transforms raw data into meaningful insights, helping in better decision-making and performance tracking.
